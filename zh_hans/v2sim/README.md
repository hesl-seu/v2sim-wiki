# V2Sim

本页面适用于V2Sim：考虑车网互动(V2G)城市**微观**交通网与配电网耦联合仿真平台。如需使用V2Sim-UX，请访问[此处](v2simux/)。([代码](https://github.com/hesl-seu/v2sim/))
