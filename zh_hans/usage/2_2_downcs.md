# 下载充电站

V2Sim 能够自动从高德地图下载电动汽车充电站。但是，您需要自行准备一个密钥。目前，在高德地图申请密钥是免费的。请访问[官方网站](https://lbs.amap.com/)完成此操作。

**注意：由于服务提供商的限制，本节可能仅适用于中国大陆用户。某些服务仅提供简体中文版本。**

**免责声明：** 如果您在 V2Sim 中使用 `下载充电站` 功能，即视为您同意以下声明：
+ V2Sim 不会收集您的密钥，您的密钥存储在本地。
+ 您必须同意高德地图提供的最终用户许可协议才能使用该服务。
+ V2Sim 及其开发者在任何情况下均不对申请密钥的滥用负责。

## 申请密钥
如果您没有账户，请先创建一个账户。登录控制台。在左侧边栏中，点击"应用管理"——"我的应用"查看您的应用。

点击右上角的蓝色按钮"创建新应用"。输入"应用名称"和"应用类型"，然后点击"新建"。在新创建的应用中，点击右上角的"添加Key"。输入"Key名称"并为"服务平台"选择"Web服务"。

然后您将获得一个由数字和字母组成的密钥。将此密钥输入到 V2Sim 主 GUI 的"下载充电站"页面中。

## 在 V2Sim 中下载充电站
+ 切换到 `充电站下载器` 页面，在给定的输入框中键入您的高德地图开发者密钥。
+ 点击 `下载` 从高德地图获取充电站位置。下载充电站位置时请耐心等待。该过程是自动的，请查看命令行提示符以了解进度。
  - 注意：您必须有足够的 POI 搜索配额。免费密钥的配额可能非常低，因此您可能无法下载选定区域内的所有充电站。

+ 下载充电站位置后，请关闭项目编辑器并重新打开它，以避免编辑器中任何潜在的错误。

## 额外说明
1. 下载过程中可能会发生错误。在这种情况下，请将 V2Sim 根文件夹中的 `buf.csv` 文件复制到您的案例中，并将其重命名为 `cs.csv`。它包含已下载充电站的缓存。

2. 如果您使用的是 V2Sim-GUI，您可能需要重新打开 GUI 才能查看下载的充电站。
