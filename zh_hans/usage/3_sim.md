+ **开始时间**：仿真的开始时间。
+ **结束时间**：仿真的结束时间。
+ (仅 V2Sim-UX) **中断时间**：仿真的实际结束时间。此参数用于从保存的状态恢复仿真。对于 V2Sim-UX，当加载保存的状态时，其参数无法修改。
+ **时间步长**：V2Sim 的仿真步长。
+ **随机种子**：V2Sim 的随机化种子。理论上，相同的种子将产生相同的结果。
+ **加载状态**：是否加载现有的保存状态
    - 从不：不加载任何状态。
    - 从上次结果：加载上次仿真中保存的状态（`<案例文件夹>/results/saved_state`）。
    - 从案例文件夹：加载案例文件夹中保存的状态（`<案例文件夹>/saved_state`）。
+ **保存状态**：选择保存状态的时机
    - 当中止时：当仿真被 Ctrl+C 中止时。
    - 当仿真结束时：当仿真正常结束时。
    - 当中止或结束时将状态复制到案例文件夹：当仿真结束或中止时，将状态文件夹从 `<案例文件夹>/results/saved_state` 复制到 `<案例文件夹>/saved_state`。

+ (仅 V2Sim) **强制使用路由缓存**

+ (仅 V2Sim) **可视化道路网络**：勾选此选项以在仿真时显示 SUMO 图形用户界面。**记得点击 SUMO 图形用户界面工具栏中的 "Play" 按钮开始仿真！**

+ (仅 V2Sim-UX) **禁用并行仿真**：当您使用 Python 3.14t 进行并行 V2Sim-UX 时，可以通过勾选此选项强制其行为像非并行 V2Sim-UX 一样。

+ (仅 V2Sim-UX) **显示 UXSim 进度**：勾选此选项以显示 UXSim 的进度而不是 V2Sim-UX 的进度。

+ (仅 V2Sim-UX) **随机化 UXSim 交通**：勾选此选项以在 UXSim 中使用非确定性流合并。

+ **路径规划算法**：用于在起点和终点之间查找路径的算法。候选算法有：`dijkstra` 和 `astar`，对于 V2Sim，还有额外的候选算法 `CH` 和 `CHWrapper`。

+ **重要提示**：`基本信息` 中的参数在点击 `开始仿真！` 之前不会被保存！`插件` 中的 `保存` 按钮仅保存插件信息。

如果您使用的是 `v2sim-gui` 或 `v2simux-gui`，请确保左侧列中的 `快充站`、`慢充站`、`道路网络` 和 `车辆` 不是 `None`。然后返回 `仿真` 页面，勾选您想要的统计项，并点击 `开始仿真！`。

一旦您点击了 `开始仿真！`，窗口将关闭。请在仿真期间耐心等待。当仿真一个大型真实世界网络时，可能会花费数小时。您可以在命令提示符中查看进度和预计所需时间。
