# 充电站生成程序

命令：

```
v2sim-gen-cs \
    -d <项目文件夹> \
    [--type <scs|fcs>] \
    [--slots <充电桩数量>] \
    [--pbuy <购买价格>] \
    [--randomize-pbuy] \
    [--psell <出售价格>] \
    [--randomize-psell] \
    [--seed <随机种子>] \
    [--n-cs <充电站数量>] \
    [--cs-names <充电站名称>] \
    [--n-bus <母线数量>] \
    [--bus-names <使用的母线>] \
```

+ d: 项目文件夹
+ type: 充电站类型，默认为快充站，可选快充站或慢充站
+ slots: 充电桩数量，默认为 10
+ pbuy: 购买价格，默认为 1.5 元/千瓦时
+ randomize-pbuy: 随机化购买价格
+ psell: 出售价格，默认为 1 元/千瓦时
+ randomize-psell: 随机化出售价格
+ seed: 随机种子。默认值：当前时间的纳秒值
+ 充电站选择：
  - 默认情况下，使用所有可用的充电站，可通过以下选项更改：
  - n-cs: 快充站数量，指定此选项将使快充站的选择变为随机
  - cs-names: 快充站名称，指定此选项将使快充站的选择变为指定
  - 以上两个选项不能同时使用
+ 母线选择：
  - 默认情况下，使用所有母线，可通过以下选项更改：
  - n-bus: 使用的母线数量，指定此选项将使母线的选择变为随机
  - bus-names: 使用的母线，指定此选项将使母线的选择变为指定
  - 以上两个选项不能同时使用
