# 多进程并行仿真工具

<details>
<summary>当前版本尚未提供该工具。以下隐藏信息仅供旧版使用。</summary>
命令：

```
v2sim-para \
    [-p <最大并行任务数>] \
    [-r <输出文件夹>] \
    [-f <命令行文件>] \
    [-c <仿真参数命令行>] \
    [-n <仿真次数>]
```

+ p: 最大并行任务数，默认为"CPU核心数 - 1"
+ r: 输出文件夹，默认为 results_set
+ 以下两组参数互斥：
+ 第一组：
  - f: 从文件中读取仿真参数，该文件的每一行是一个仿真参数命令行，仿真参数命令行的格式如下
+ 第二组：
  - c: 仿真参数命令行，格式与 main.py 的参数相同，但必须省略所有 '-seed' 参数（'-seed' 参数由并行仿真器自动分配）
  - n: 仿真次数，默认为 50
    
+ 参数示例：
  - `-p 7 -d 37nodes -r results_set -c "-gen-veh '-n 5000' -gen-fcs '-pbuy 1.5 -slots 10' -gen-scs '-pbuy 1.5 -slots 10'" -n 50`
